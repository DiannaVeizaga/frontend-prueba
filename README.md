## Frontend Prueba##

Aplicación frontend desarrollada con React y Vite que permite visualizar publicaciones, filtrar por usuario, buscar contenido y ver el detalle de cada publicación.

## Requisitos##

- Tener instalado Node.js

## Instalación ##

1. Clonar el repositorio

git clone https://github.com/DiannaVeizaga/frontend-prueba.git

2. Entrar a la carpeta del proyecto

cd frontend-prueba

3. Instalar dependencias

npm install

4. Ejecutar el proyecto

npm run dev

## Acceso##

Abrir en el navegador:

http://localhost:5173
